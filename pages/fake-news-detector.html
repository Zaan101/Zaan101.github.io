Project Overview

A machine learning project that classifies news headlines as real or fake using NLP. It includes a FastAPI service for real-time inference and a small UI to try examples.

Languages and Libraries Used

  • Languages: Python, HTML
  • ML/NLP: PyTorch (BiLSTM), NLTK, GloVe embeddings
  • Data: pandas, numpy
  • Web/API: FastAPI, Uvicorn
  • Visualization: Plotly (with UMAP/embedding views)

Features

  • Text preprocessing (tokenization, cleaning) and GloVe-based embeddings
  • BiLSTM model for binary classification (fake vs. real)
  • FastAPI endpoint for real-time predictions (+ interactive form)
  • Training/evaluation scripts and saved weights/checkpoints
  • Embedding visualization to explore nearest neighbors
